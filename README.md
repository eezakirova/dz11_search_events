**Тема:** Python для аналитиков ИБ: поиск негативных событий  

---

## Задание

Проанализировать два типа логов из представленного набора данных, найти подозрительные события и визуализировать 10 наиболее распространённых подозрительных событий.

---

## Этап 1. Загрузка и подготовка данных

1. Скачайте или используйте предоставленный набор данных `botsv1`, содержащий логи **WinEventLog** и **DNS**.
2. Загрузите данные в датафреймы с помощью **Pandas**.
3. При необходимости выполните нормализацию данных:
   - приведение к единому формату;
   - очистку данных.

---

## Этап 2. Анализ данных

1. **WinEventLog**:
   - Найдите подозрительные события по полю `EventID`;
   - Примеры: входы с ошибками, эскалация привилегий и другие подозрительные действия.

2. **DNS-логи**:
   - Определите потенциально вредоносные или подозрительные запросы;
   - Примеры: частые обращения к редким доменам, нестандартные поддомены.

---

## Этап 3. Визуализация данных

1. Постройте график **топ-10 подозрительных событий**.
2. Возможные варианты визуализации:
   - отдельные диаграммы для **WinEventLog** и **DNS**;
   - объединённая визуализация.
3. Используйте **Seaborn** или **Matplotlib** для создания графиков.

---

## Результат

В результате выполнения задания вы получите:

- датафреймы с нормализованными логами;
- выявленные подозрительные события;
- визуализацию топ-10 событий, которая поможет в аналитике ИБ.
